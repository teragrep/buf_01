Detailed Description

Decorator for MemorySegmentContainer that zeroes the encapsulated MemorySegment on close().

Starts with one initial reference. Internally uses a NonTerminatingPhaser to track reference count in a non-blocking way.

Responsibilities:
  • Reference tracking
  • Provides access to encapsulated MemorySegment
  • Zero references removes access to MemorySegment
  • Fills MemorySegment with zeroes (0) on close
Collaborators:
  • Lease
  • Pool
  • MemorySegmentSubLease
